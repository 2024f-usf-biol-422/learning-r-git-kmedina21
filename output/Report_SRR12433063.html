Project Description

This project analyzes single nucleotide polymorphism (SNP) data from SARS-CoV-2 samples. It subsets the data to focus solely on one sample only. The SNP data is then visualized in a plot. The first package used is dplyr (Wickham et al., 2023). Then ggplot2 is used to create the plot (Wickham, 2016). Lastly, knitr is used to create the report (Xie, 2023).
